🚀 快速开始

创建新活动

  1. 打开添加活动对话框 菜单栏选择 "文件 → 添加活动" 或使用快捷键 Ctrl+N
  2. 填写活动信息 输入活动名称，选择颜色和图标，可设置每日目标时长
  3. 保存活动 点击确定按钮，新活动将出现在左侧活动控制区

开始计时

  1. 选择活动 在左侧"活动控制"区找到要开始的活动
  2. 点击开始按钮 点击活动卡片上的 ▶ 按钮开始计时
  3. 添加备注（可选） 在弹出的对话框中输入本次活动的备注
  4. 查看运行状态 运行中的活动会显示在"运行中的活动"区域，实时显示计时

管理运行中的活动

  1. 暂停活动 点击 ⏸ 按钮暂停计时，时间不再增长
  2. 继续活动 点击 ▶ 按钮继续计时
  3. 停止活动 点击 ⏹ 按钮结束活动，记录将保存到日志
  4. 编辑活动 点击 ✏ 按钮可修改备注或调整时间

📤 导出报告

  1. 切换到报告导出标签页 点击主界面的"📋 报告导出"标签
  2. 选择日期范围 设置报告的开始和结束日期
  3. 选择导出格式 支持 Excel (.xlsx) 和 CSV 格式
  4. 点击导出按钮 选择保存位置，报告将自动生成

❓ 常见问题

  • OpenTimeLog 支持同时运行多个活动。只需在活动控制区点击不同活动的开始按钮， 它们会同时计时并显示在"运行中的活动"区域。适合需要同时处理多项任务的场景， 如一边听课一边做笔记。
  • 可以！使用"手动添加记录"功能（快捷键 Ctrl+M）。在弹出的对话框中选择活动类型， 设置开始时间和结束时间，即可补录该时间段的活动记录。
  • 不会丢失。退出程序时，如果有运行中的活动，会弹出提示询问是否停止这些活动。 您可以选择停止并保存，或取消退出继续计时。此外，程序默认关闭时会最小化到系统托盘， 活动会继续运行。
  • 在"日志记录"标签页中找到需要删除的记录，点击记录右侧的删除按钮（红色垃圾桶图标）， 确认后即可删除。您也可以选择编辑记录来修改时间或备注。
  • 请检查以下几点：
    1. 确保程序正在运行（可以最小化到托盘，但不能完全退出）
    2. 日程的活动名称需要与已创建的活动完全匹配
    3. 如果该活动已经在运行中，系统不会重复提醒
    4. 检查系统通知设置是否允许应用弹出通知
  • 在菜单栏选择"主题"，可以选择三种预设主题：
    • 默认主题 - 清爽的浅色主题
    • 现代科技 (男) - 深色科技感主题
    • 柔和优美 (女) - 温柔的粉色主题
    主题设置会自动保存，下次启动时自动应用。
  • 所有数据存储在程序目录下的 time_logger.db 文件中（SQLite 数据库）。 要备份数据，只需复制此文件到安全的位置。恢复时将备份文件放回原位置即可。 建议定期导出报告作为额外的数据备份。
